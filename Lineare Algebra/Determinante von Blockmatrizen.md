# Einheitsblöcke
Falls
$$\Gamma = \begin{pmatrix}I & 0 \\ 0 & A\end{pmatrix} \vee \Gamma = \begin{pmatrix}A & 0 \\ 0 & I\end{pmatrix}$$
dann gilt
$$\implies \det(\Gamma) = \det(A)$$
# Trianguläre
Falls
$$\Gamma = \begin{pmatrix}A & B \\ 0 & D\end{pmatrix}$$
dann gilt
$$\implies \det(\Gamma) = \det(A)\det(D)$$
