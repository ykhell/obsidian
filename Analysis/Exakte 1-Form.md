# Definition
Eine stetige [[1-Form]] $\omega : U \to V^*$ hei√üt **exakt**, wenn eine $C^1$-Funktion $f: U \to \Bbb{R}$ existiert, sodass $df = \omega$, wir nennen in diesem Fall $f$ ein [[Potential]] von $\omega$.

# Bemerkung
Ist $\omega$ exakt, so ist $\omega$ auch [[Geschlossene 1-Form|geschlossen]] (symmetrische [[Jacobi-Matrix]]).
