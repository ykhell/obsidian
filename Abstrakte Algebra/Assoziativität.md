# Definition
Eine binäre Verknüpfung $*: A\to A$ auf einer [[Menge]] $A$ heißt assoziativ, wenn für alle $a, b, c \in A$ das Assoziativgesetz
		$a*(b*c) = (a*b)*c$
gilt. Die Klammern spielen dann keine Rolle und können weggelassen werden.
